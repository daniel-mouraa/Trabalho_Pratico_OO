Trabalho prático final da matéria Orientação a Objetos, materia ministrada pelo Prof. Andre Lanna da Universidade de Brasilia (UnB).
  A proposta incial do trabalho envolvia a criação de um Sistema de Vendas e Fidelidade para uma cafeteria, no qual além de possibilitar o cadastro
  de pedidos e de um fluxo de vendar, permitisse a gestão de produtos no cardapio e um sistema de cashback para os clientes já fidelizados. Utilizamos
  dos conceitos de Orientação a Objetos, como diferentes classes, pacotes, herança entre algumas classes, polimorfismo, dentre outras.
